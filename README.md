# Movies Firestore Dashboard

Dashboard interactivo en Streamlit conectado a Cloud Firestore.

## Funcionalidades
- Mostrar todos los filmes
- Buscar por título
- Filtrar por director
- Insertar un nuevo filme

## Tecnologías
Python, Streamlit, Firebase Admin SDK, Cloud Firestore

## Deploy
La app está desplegada en Streamlit Community Cloud.
